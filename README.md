# Renters Hub

Free, plain-English tools and guides for private renters in England under the Renters' Rights Act 2025.

## Getting started

```powershell
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build for production

```powershell
npm run build
npm run start
```

## Stack

Next.js 16 (App Router), TypeScript, Tailwind CSS 4, Fraunces + Geist via next/font, lucide-react.

No database, auth, billing, or LLM calls in v0 — everything is stateless and runs in the browser.

See `CLAUDE.md` for project context and conventions.
